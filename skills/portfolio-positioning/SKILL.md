# Portfolio Positioning Skill

Use this skill when preparing the project for GitHub, LinkedIn, CV, or interviews.

## When to use
Use when the task involves:
- README improvement
- Project pitch
- LinkedIn post
- CV bullets
- Interview explanation
- Demo script
- Architecture summary

## Principles
- Sell backend value, not just AI usage.
- Emphasize responsible AI.
- Emphasize deterministic rules, auditability, validation, tests, and maintainability.
- Avoid sounding like a simple ChatGPT wrapper.
- Explain business value clearly.

## Output
Return:
- Short pitch
- Technical pitch
- Business value
- Backend skills demonstrated
- AI skills demonstrated
- Interview talking points
- Suggested improvements
